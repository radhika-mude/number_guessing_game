# guessing game

A number guessing game built in Python.

## Programs Written

### guessing_game.py
Asks the user to guess a secret number between 1 and 10. 
Gives "too high" or "too low" hints until the correct answer 
is entered. Tracks total number of attempts.
- **Concepts used:** while loop, if/else, int(), input(), counter
